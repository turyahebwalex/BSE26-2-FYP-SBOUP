# CPU-only PyTorch, matching the cv-generation / learning-engine services and
# the SDD's "torch CPU" image budget. The chatbot only runs CPU inference
# (sentence-transformers for the ChromaDB knowledge base), so the CUDA-bundled
# default torch was pure dead weight — it ballooned the image to ~9.4GB and made
# every rebuild re-download 500MB+ of GPU libraries. Pinning the CPU wheel keeps
# the image near ~2GB. Must precede sentence-transformers so pip resolves CPU
# torch first instead of pulling the CUDA default as a transitive dependency.
--extra-index-url https://download.pytorch.org/whl/cpu
